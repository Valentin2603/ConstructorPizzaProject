# Pizza Constructor Project

Консольное приложение для создания и управления пиццами и заказами.

## Возможности
- CRUD ингредиентов
- CRUD основ
- CRUD пицц
- Бортики
- Комбинированные пиццы
- Размеры
- Заказы
- Отложенные заказы
- Фильтрация заказов
- UUID для заказов
- Stream API и Comparator

## Технологии
- Java 17
- OOP
- Stream API
- UUID
